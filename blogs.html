Question-2: What is the difference between innerHTML, innerText, and textContent ?

Difference between innerHTML, innerText, and textContent:
1. innerHtml = returns all the text along with the html tag and hidden text or comment inside an element.
2. innText = only returns the visible text inside an element
3. textContent = returns both visible and hidden texts only of inside an element

Question-3: What is event delegation in the DOM?

Event Delegation:
The process of adding event listener in parent of elements instead of adding event in every child element is called event delegation.

since a triggered element returns to its parent element anyways, its more efficiant to add event listener in parent for capturing the event.target .

Event Delegation is used for:
1. Less memory consupmtion
2. no need to add event listener to all child element
etc.

Question-4: What is event bubbling in the DOM?

Event Bubbling:
When an event is triggered, after capturing the event target the event get propagate or returns to the parent element and triggers the parent element. and again it propagates to the next parent element and triggers it all the way to root element.

This process is call event bubbling, where all the parent elements of the event gets triggered to the root element. One by one.

Question-5: How do you create, add, and remove elements using JavaScript?

Create, Add and remove elements:

1. first create an element using document.createElement('tag'). Then add innerText ot innerHtml of the the newly created element.
2. using element.append(child) where child = new element, we can add the new element in the document where we want.
3. to remove element we can use this methods
 a. parent.remove() removes all the child elements of an element
 b. parent.removeChild(child) to remove a specific element
